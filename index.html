Artifact 1: AI-Powered Finance Assistants — Applying Design Thinking to Build Two Chatbots

Introduction As part of my AI/ML coursework, I completed a hands-on AI Lab where I practiced with generative AI tools and then designed, built, and tested two chatbots grounded in my professional background in distribution finance.
Description I built two Chatbase-hosted AI agents using the design thinking framework (empathy, define, ideate, prototype, test): first, a Distribution Finance Reporting Assistant that explains financial terms like shrink, spoilage, and inflation in plain language; then a more advanced Variance Investigation Assistant that guides a user through a structured, multi-step diagnostic process to identify the likely root cause of a financial metric change.
Objective To apply generative AI tools and design thinking methodology to real problems in financial reporting and analysis — moving from simple Q&A to more complex, sequential reasoning as my skills developed through the lab.
Process I started by practicing with multiple LLMs and a custom GPT to understand how prompt design and context affect output quality. I then applied design thinking to plan each bot's objective, scope, and constraints before building anything. The first bot used single-turn Q&A; the second required explicit sequencing logic so the AI would ask one diagnostic question at a time and build toward a conclusion. I tested both extensively, including with unplanned questions and deliberately ambiguous input.
Tools and Technologies Used Chatbase (AI agent builder), GPT-5.4 Mini, prompt engineering, design thinking methodology.
Value Proposition This project demonstrates my ability to translate real financial domain expertise into practical, well-scoped AI tools — going beyond just using AI to actually designing how it should behave and how to test it rigorously.
Unique Value Unlike a generic coursework exercise, both bots are grounded in real distribution finance concepts I work with professionally, and the progression from a simple to a more complex build shows growth in applying AI/ML concepts over a single assignment.
Relevance Directly relevant to FP&A, financial systems, and business analytics roles, where combining finance domain knowledge with practical AI/data tooling is an increasingly valued skill.
